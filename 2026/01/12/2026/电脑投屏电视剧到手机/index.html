电脑局域网投屏电视剧到手机

天气寒冷，适合躺在被窝里面看电视剧，但是资源其实都是在电脑上下载的，那么就存在将其投屏到手机上观看的需求，需要花点时间解决一下下。

使用Jellyfin

项目地址：https://github.com/jellyfin/jellyfin

1、电脑上安装Jellyfin后端服务

后端服务：https://jellyfin.org/downloads/windows/

里面有一个Jellyfin.Windows.Tray.exe托盘小程序，从里面可以启动服务，之后访问主页http://localhost:8096/配置就行了

安装过程中遇到的问题

1、系统上可能需要安装.NET 9.0 Runtime ? 懒得测试了

2、启动进程提示：

问题原因：

Jellyfin 所使用的 .NET 运行库与较新的 Intel CPU（通常是 11代及以上）和 Windows 安全机制（CET/影子堆栈）发生了冲突。

解决方案：

修改一下 Windows 的安全设置：

  1. 点击 Windows 开始菜单，搜索并打开 “漏洞利用保护” (英文系统搜 Exploit Protection)。
  2. 在打开的窗口中，点击顶部的 “程序设置” 选项卡（默认是在“系统设置”）。
  3. 添加 Jellyfin应用程序
  4. 关闭“硬件强制堆栈保护” (Hardware-enforced Stack Protection)。
  5. 再次尝试启动
