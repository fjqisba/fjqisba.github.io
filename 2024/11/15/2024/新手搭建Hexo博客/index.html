一个项目是我的私有项目MyBlog，用于存储真实的hexo博客项目，存储加密文章原始内容，并通过workflows发布，要做到这样，还得进行以下步骤

1、去Github个人Settings -> Developer Settings -> Personal access tokens -> Fine-grained tokens，去生成一个token，这个token得具备操作fjqisba.github.io项目的权力，生成好后记住这个token值

2、去私有项目MyBlog -> Settings -> Security -> Secrets and variables -> Actions -> 建立一个Secrets变量X_BLOG_SITE，值就填之前生成的token就行了。
